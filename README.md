This is a 100% offline fake-news detection app built with Streamlit.
It uses TF-IDF + Logistic Regression (local ML model) and includes:

- Full article scoring
- Sentence-level fake probability
- Heatmap visualization
- PDF support
- URL extraction
- Charts + CSV download

## Run locally

pip install -r requirements.txt  
streamlit run app.py  

## Deploy on Streamlit Cloud, Render, HuggingFace Spaces
The app is 100% offline and needs no API keys.
